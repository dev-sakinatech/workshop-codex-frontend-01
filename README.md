# eSamsat Maluku Utara Prototype

Dokumentasi dan prototipe HTML untuk aplikasi eSamsat yang memusatkan penetapan dan penagihan pajak kendaraan bermotor di Bapenda Maluku Utara.

## Struktur
- `docs/fsd.md`: Analisis dan rancangan fitur.
- `docs/features.md`: Daftar fitur yang diturunkan dari FSD.
- `docs/index.html`: Prototipe statis dengan layout profesional dan peta fitur di sidebar.

## Cara Menjalankan Preview
1. Jalankan server statis dari root repo:
   ```bash
   python -m http.server 8000
   ```
2. Buka browser ke `http://localhost:8000/docs/index.html`.

## Catatan
Prototipe ini tidak memiliki backend; seluruh data bersifat statis untuk mendukung demonstrasi UI/UX awal.
